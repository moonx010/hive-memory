# S-Tier Roadmap — Enterprise Knowledge Management Platform

## Market Context
- Enterprise KM market: $7.66B (2025) → $51.36B (2030), 46.7% CAGR
- Target position: Developer-first, local-first, agent-native enterprise knowledge layer
- Unoccupied niche: MCP-native + data sovereignty + enterprise-grade

## Phase 1: Team Scale (0-6 months) — A → A+
1. Temporal entity validity (valid_from/to/superseded_by)
2. Webhook-based connectors (Slack Events API, <30s ingestion)
3. Query rewriting + intent parsing (LLM pre-retrieval)
4. Agentic multi-step retrieval (planner-retriever-synthesizer)
5. Conflict resolution surfacing

## Phase 2: Enterprise (6-12 months) — A+ → S
1. Multi-tenancy (org→workspace→user)
2. SAML SSO + SCIM (via WorkOS)
3. Permission-enforced retrieval (Glean model)
4. Community detection + GraphRAG summaries (Leiden algorithm)
5. SOC2 Type II compliance
6. Data residency (EU/US/APAC)

## Phase 3: Platform (12-24 months) — S → S+
1. Self-corrective RAG (CRAG)
2. CDC streaming ingestion (<5s)
3. 100+ connectors (marketplace)
4. MCP Gateway (enterprise auth + audit)
5. ISO 42001 / EU AI Act compliance
6. Federated knowledge sharing

## Competitive Moats
- MCP-native (97M+ monthly SDK downloads)
- Neuroscience-inspired graph (15 axon types, Hebbian LTP/LTD)
- Local-first (73% enterprise priority: data sovereignty)
- Meeting intelligence (unique in market segment)
